# Personal pronouns

In Zerita, only the first 3 persons have a personal pronoun.
The 4th person is used for expressing all the objects or none of them,
therefore they are covered by the indefinite pronouns.

| Person | 1st |
| ------ | --- |
| 1st    | me  |
| 2nd    | ve  |
| 3rd    | asé |

## Declination tables

The personal pronouns follow the declination rules of the nouns.

### 1st person

| Case/Number | Singular | Plural | Plural (inc) |
| ----------- | -------- | ------ | ------------ |
| Nominative  | meis     | mesis  | emesis       |
| Accusative  | meon     | meson  | emeson       |
| Dative      | meim     | mesim  | emesim       |
| Thematic    | mei      | mesi   | emesi        |
| Absolutive  | me       | mese   | emese        |

### 2nd person

| Case/Number | Singular | Plural |
| ----------- | -------- | ------ |
| Nominative  | veis     | vesis  |
| Accusative  | veon     | veson  |
| Dative      | veim     | vesim  |
| Thematic    | vei      | vesi   |
| Absolutive  | ve       | vese   |

### 3rd person

| Case/Number | Singular | Fractional | Plural |
| ----------- | -------- | ---------- | ------ |
| Nominative  | aseis    | asedis     | asesis |
| Accusative  | aseon    | asedon     | aseson |
| Dative      | aseim    | asedim     | asesim |
| Thematic    | asei     | asedi      | asesi  |
| Absolutive  | asé      | asede      | asese  |
