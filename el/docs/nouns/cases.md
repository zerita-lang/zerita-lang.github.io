# Πτώσεις

Η ζερίτα έχει 5 πτώσεις, που πάλι δηλώνονται με επιθήματα. Θα χρησιμοποιήσουμε το "ros",
που σημαίνει "τριαντάφυλλο".

- **Ονομαστική:** επίθημα -is, χρησιμοποιείται για να δηλώσει το υποκείμενο της πρότασης: rosis.
- **Αιτιατική:** επίθημα -on, χρησιμοποιείται για να δηλώσει το άμεσο αντικείμενο της πρότασης: roson.
- **Δοτική:** επίθημα -im, χρησιμοποιείται για να δηλώσει το έμμεσο αντικείμενο της πρότασης: rosim.
- **Θεματική:** επίθημα -i, χρησιμοποιείται για να δηλώσει το θέμα της πρότασης: rosi.
- **Απόλυτη:** επίθημα -e, χρησιμοποιείται μετά από προθέσεις και για να απευθυνθείς σε κάποιον: rose.
