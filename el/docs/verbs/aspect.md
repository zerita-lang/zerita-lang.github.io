# Όψη

Στη ζερίτα υπάρχουν 9 όψεις, οι οποίες χρησιμοποιούνται για να
υποδείξουν το χρόνο της ενέργειας του ρήματος.
Υπάρχουν τρεις τριάδες από αυτές.
Κάθε τριάδα περιέχει μια απλή, μια τετελεσμένη και μια αναμενόμενη όψη

| Διάρκεια/Τελεστικότητα | Απλή | Τετελεσμένη | Αναμενόμενη |
| ---------------------- | ---- | ----------- | ----------- |
| Απλή                   |      | -il         | -ol         |
| Διαρκείας              | -um  | -ilum       | -olum       |
| Συνήθειας              | -em  | -ilem       | -olem       |

## Παραδείγματα

- _potivo e akvon:_
    - Ήπια νερό
- _potivilo e akvon:_
    - Έιχα πιει νερό
- _potivolo e akvon:_
    - Θα ξεκινούσα να πιω νερό
- _potivumo e akvon:_
    - Έπινα νερό
- _potivilumo e akvon:_
    - Είχα τελειώσει να πίνω νερό
- _potivolumo e akvon:_
    - Θα ξεκινούσα να πίνω νερό
- _potivemo e akvon:_
    - Είχα τη συνήθεια να πίνω νερό
- _potivilemo e akvon:_
    - Είχα πιο πριν τη συνήθεια να πίνω νερό
- _potivolemo e akvon:_
    - Θα ξεκινούσα να έχω τη συνήθεια να πίνω νερό
