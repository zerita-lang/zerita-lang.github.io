# Πρόσωπο

Στη ζερίτα, υπάρχουν 4 γραμματικά πρόσωπα: 1ο, 2ο, 3ο και 4ο.
Το 4ο πρόσωπο χρησιμοποιείται για να περιγράψει συνολικές περιπτώσεις, δηλαδή τον μηδενικό και τον ολικό αριθμό.
Επιπλέον, το 1ο πρόσωπο πληθυντικού έχει δύο μορφές: την συμπεριληπτική και την αποκλειστική.

Τα επιθήματα είναι:

| Πρόσωπο/Αριθμός | Ενικός | Κλασματικός | Πληθυντικός | Μηδενικός | Ολικός |
| --------------- | ------ | ----------- | ----------- | --------- | ------ |
| 1ο              | -o     |             | -os         |           |        |
| 1ο (συμπ)       |        |             | -as         |           |        |
| 2ο              | -i     |             | -is         |           |        |
| 3ο              | -e     | -ed         | -es         |           |        |
| 4ο              |        |             |             | -un       | -ur    |

Η ζερίτα είναι μια γλώσσα που παραλείπεται ότι εννοείται από τα συμφραζόμενα, οπότε οι αντωνυμίες δεν είναι απαραίτητες και θα πρέπει να χρησιμοποιούνται μόνο αν απαιτείται έμφαση.
