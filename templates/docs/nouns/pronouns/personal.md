# <x-trans>Personal pronouns</x-trans>

<x-trans>In Zerita, only the first 3 persons have a personal pronoun.</x-trans>
<x-trans>The 4th person is used for expressing all the objects or none of them,
therefore they are covered by the indefinite pronouns.</x-trans>

| <x-trans>Person</x-trans> | <x-trans>Pronoun</x-trans> |
| ------------------------- | -------------------------- |
| 1st                       | me                         |
| 2nd                       | ve                         |
| 3rd                       | asé                        |
